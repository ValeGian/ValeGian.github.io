# me.json

Myself as a JSON document.

I use this file to generate my [personal site](https://valegian.github.io). If you like it, you can check the code [here](https://github.com/ValeGian/ValeGian.github.io)

## Structure

* `anagraphic`
  * `fullname`
    * `first` First Name
    * `last` Last name
  * `birthday`
    * `day`
    * `month`
    * `year`
  * `nationality`
  * `picture`
  * `bio`
  * `qualifications`
